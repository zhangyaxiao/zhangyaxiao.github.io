2018.07.18  热度 ℃

Java 8 Lambda 表达式

Lamdba 表达式是java8推出的新特性，它允许把函数作为一个方法的参数（函数作为参数传递进方法中）

语法

  • 可选类型声明： 不需要声明参数类型，编译器可以统一识别参数组
  • 可选的参数圆括号： 一个参数无需定义圆括号，但多个参数需要定义圆括号
  • 可选的大括号： 如果主体包含一个语句，就不需要使用大括号
  • 可选的返回关键字： 如果主体只有一个表达式返回值则编译器会自动返回值，大括号需要指明表达式返回一个数值
